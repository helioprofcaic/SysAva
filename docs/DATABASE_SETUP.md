# 🗄️ Guia de Configuração do Banco de Dados (Supabase)

Siga estes passos para criar as tabelas necessárias no seu banco de dados.

## 1. Acessar o Editor SQL
1. Entre no seu projeto no [Supabase](https://supabase.com).
2. No menu lateral esquerdo, clique em **SQL Editor** (ícone de terminal `>_`).
3. Clique em **New Query** (ou use uma aba em branco).

## 2. Criar Tabela do Fórum
Copie o código abaixo, cole no editor e clique no botão **Run** (ou pressione `Ctrl+Enter`):

```sql
-- Tabela para armazenar mensagens do fórum
create table forum_posts (
  id bigint generated by default as identity primary key,
  created_at timestamp with time zone default timezone('utc'::text, now()) not null,
  user_name text,
  message text
);
```

## 3. Criar Tabela de Usuários (Login)
Apague o código anterior do editor, cole este novo código e clique em **Run**:

```sql
-- Tabela para armazenar usuários e senhas
create table app_users (
  username text primary key,
  password text not null, -- Armazenará a senha criptografada (hash)
  ra text unique, -- Registro do Aluno
  name text,
  role text default 'student' -- Papel do usuário: 'student' ou 'admin'
);
```

## 4. Criar Usuário de Teste
Para conseguir entrar no sistema imediatamente, crie um usuário inicial rodando este comando:

```sql
-- Inserir aluno teste (Usuário: aluno, Senha: 123456)
insert into app_users (username, password, name) 
values ('aluno', '123456', 'Aluno Exemplo');
```

## 5. Onde encontrar as Chaves (URL e Key)
1. No menu lateral, clique em **Project Settings** (ícone de engrenagem ⚙️).
2. Clique na seção **API**.
3. Você precisará copiar os valores de **Project URL** e **anon public key** para colocar no seu arquivo de segredos (`secrets.toml` ou no Streamlit Cloud).
